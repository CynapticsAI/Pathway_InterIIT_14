# Pathway's Agentic System

Agentic System implemented in pathway using kafka for agent to agent communication.

### Agents:
    - Clarifier Agent: Asks user for further clarification in case query is vague.
    - Orchestrator Agent: Orchestrates the flow of queries among agents and can ask for further clarification to any agent.
    - Market Analyzer Agent: Can answer question related to current market affairs.
    - Macro Agent: Answers queries related to macro-economics and has access to sector rankings generated by our MCP tools.
    - Portfolio Agent: Can diversify, rebalance or create the user's portfolio from scratch using mathematical algorithms and data from other agents offering utmost transparency
    - Strategy Agent: Tests and creates strategies implemented via a Genetic Algorithm. Back testing is done via historical data stored in our database
